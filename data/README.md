# Dataset
This project uses the following public datasets:

## RAVDESS
Ryerson Audio-Visual Database of Emotional Speech and Song

Official Download:
https://zenodo.org/records/1188976

---

## CREMA-D
Crowd-sourced Emotional Multimodal Actors Dataset

Official Download:
https://github.com/CheyneyComputerScience/CREMA-D

---

## Notes
The datasets are **not included** in this repository due to licensing terms and repository size limitations.

Please download the datasets from their official sources and place them inside the `data/` directory before running the project.

Expected directory structure:

data/
├── RAVDESS/
└── CREMA-D/
